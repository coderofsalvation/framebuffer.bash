Framebuffer.bash

framebuffer in bash to draw ascii pixels, borders, animate e.g. (bash ui bootstrap)

Download as .zip Download as .tar.gz View on GitHub

Why

Fullfledged html/css/javascript applications are nice, but sometimes a simple email with asciitext does the job. This is for minimalists and productivity fanatics.

plugins/Barchart

           ██

code examples:

put 2 3  "stock   :"; echo "0,1,2,0,2,1,0,1,2,1" | barchart 10 2 20 
put 2 7  "banana  :"; redis-cli --csv LRANGE foo 0 -1 | sed 's/[^0-9,\\-]*//g' | vbarchart 10 6 20
