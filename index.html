Framebuffer.bash

framebuffer in bash to draw ascii pixels, borders, animate e.g. (bash ui bootstrap)

Download as .zip Download as .tar.gz View on GitHub

Why

Fullfledged html/css/javascript applications are nice, but sometimes a simple email with asciitext does the job. This is for minimalists, productivity fanatics and unix shell maniacs.

Barcharts!

will produce something like this:

 apple   : ——▄▄▀▀——▀▀▄▄——▄▄▀▀▄▄———    

 banana  : ▁▁▂▂▃▃▄▄▅▅██▄▄▂▂▁▁▁▁▁▁▁

 plum    : ██████
 orange  : ████
 apple   : ██
 foo     : ██████████

code example:

put 2 3  "stock   :"; echo "0,1,2,0,2,1,0,1,2,1" | fb barchart 10 2 20 
put 2 7  "banana  :"; redis-cli --csv LRANGE foo 0 -1 | sed 's/[^0-9,\\-]*//g' | fb vbarchart 10 6 20

Number to bar

milk                 5                    ▆▆▆▆▆
cheese               3                    ▆▆▆
palinka              14                   ▆▆▆▆▆▆▆▆▆▆▆▆▆▆

code example:

echo -e "milk 5\ncheese 3\npalinka 14" | fb number2bar 2

Ascii tables!

Get pretty aligned, copy/paste-able output:

foo long column      2  foobie
bar                  5  barrie

code example:

cat foo.csv | fb csv2ascii
cat semicolon.csv | SEP=';' fb csv2ascii

CSV's with quoted strings are supported.

Histograms!

turns logs into:

UFW        1600 █████████████████████████ 
cfg80211:    11 █████████████ 
wlan0:        9 █████ 
ath:          8 █

code example:

dmesg | fb histogram 2 25 25

Animations!

code example:

fb animate 2 4 0.1 ". o O o"

Boxes!

┌──────────────────────────────────────────────────────────┐
│date: Mon May 19 23:43:45 CEST 2014                       │
│                                                          │
└──────────────────────────────────────────────────────────┘

code example:

fb box 1 1 20 10; fb put 2 2 "date: $(date)"
